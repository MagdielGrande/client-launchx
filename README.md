# vue-3-crud

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```

### Ejecuta

1. Run the project 
```npm run serve```

2. Go to ```http://localhost:8081/```

There you can create, read, update and delete.
